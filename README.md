## 🚀 How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/codesoft-cpp-projects.git
```

2. Navigate to the project folder:

```
cd codesoft-cpp-projects
```

3. Compile the program:

```
g++ filename.cpp -o output
```

4. Run the program:

```
./output
```
